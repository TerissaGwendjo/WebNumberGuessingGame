# 🎯 Number Guessing Game Web

A full-stack Java web application built with Spring Boot where users guess a randomly generated number.

## 🚀 Features

- 🎯 Interactive number guessing game
- 🔢 Attempt counter
- 🏆 Win message with number of tries
- 🔄 Restart game functionality
- 🎚 Difficulty levels (1–50, 1–100, 1–500)
- 🎨 Styled UI with CSS

## 🛠 Tech Stack

- Java 21
- Spring Boot 3.5.12
- Thymeleaf
- HTML/CSS

## ▶️ How to Run

### Using IntelliJ IDEA
1. Open project
2. Run `NumberGuessingGameWebApplication.java`
3. Open browser: [http://localhost:7676](http://localhost:7676)

### Using Terminal
```bash
mvn spring-boot:run
